Methodology

How the data was collected and analyzed

Data Source

All home and away statistics were collected from Sports Reference Stathead, which aggregates NCAA Division I basketball data. The data was collected using the Team Game Finder tool, which allows users to find team totals from all games in a seasonal window. The data was collected in two datasets — one for true home games and one for true away games — for all Division I teams from the 2015-2016 season to the 2024-2025 season.

Excel Filtering

The collected data contained totals for every Divison I team, but this project only focused on teams in what is considered the "Power Five Conferences." Using a reference table of all the teams in these five conferences (Big Ten, Big 12, SEC, ACC, Big East), each team was given a conference label. This conference label was then used to filter the dataset to only include teams in the Power Five Conferences. With the home and away datasets filtered down to the Power Five Conferences, each team's home and away data also needed to be merged onto one dataset. Using team name as a matching label, each team's away data was matched to the home data. The resulting dataset had a row for each Power Five team that contained its totals such as Win% and Point Margin for both home and away games within the seasonal window.

Calculations

With home and away data combined, a few key calculations were made. First, to find the average point margin for each team at home and away, the point margin column was divided by the respective number of games played either home or away. To calculate the difference in home vs. away performance on key statistics, the away column was subtracted from the home column for the given statistic. For the Win % and Average Point Margin statistics, these differences were then plotted on bar charts via Flourish and published to the page.
